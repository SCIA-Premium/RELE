Rappels¶

Le DQN apprend jouer un jeu en interagissant avec l'environnement et en essayant de maximiser la rcompense future. Il fait cela en utilisant un rseau de neurones pour estimer la fonction de valeur, qui est une estimation de la somme des rcompenses futures attendues partir de l'tat et de l'action actuels. Le DQN apprend jouer en prenant des actions qui maximisent la fonction de valeur estime et en ajustant ses paramtres en fonction de la diffrence entre la fonction de valeur estime et la cible de la fonction de valeur.

Q1: Quelle est la diffrence entre un rseau neuronal profond (DQN) et un rseau neuronal classique ?

La principale diffrence entre un rseau neuronal profond et un rseau neuronal classique est que les rseaux neuronaux profonds ont plusieurs couches caches, ce qui leur permet d'apprendre des modles plus complexes partir de donnes. Les rseaux neuronaux classiques, en revanche, ont gnralement seulement une seule couche cache, ce qui limite leur capacit apprendre des modles complexes. Les rseaux neuronaux profonds sont donc souvent plus performants que les rseaux neuronaux classiques pour des tches telles que la reconnaissance d'images ou le traitement du langage naturel.

def visualize(env, title):
    # set the initial state of the environment
    state = env.reset()
    print("Shape of the action space of environement", env.action_space.shape)
    print("Observation space of environement", env.observation_space.shape)

    # initialize an empty list to store the state history
    state_history = []

    # define the number of steps to run the simulation for
    num_steps = 1000

    # run the simulation for the specified number of steps
    for step in range(num_steps):
        # choose a random action
        action = env.action_space.sample()
    
        # take a step in the environment using the chosen action
        state, reward, done, info, _ = env.step(action)
    
        # add the new state to the state history
        state_history.append(state)
    
        # check if the simulation is done
        if done:
            break

    # extract the position and velocity data from the state history
    positions = [s[0] for s in state_history]
    velocities = [s[1] for s in state_history]

    # plot the position and velocity data
    plt.plot(positions, label='position')
    plt.plot(velocities, label='velocity')
    plt.legend()
    plt.title(f"Evolution of {title} position and velocity over time")
    plt.show()

Implémentation du DQN¶

L'architecture neuronale pour rsoudre le Deep Q-Learning (DQN) est un rseau de neurones profond. Il est compos d'une couche d'entre, d'une couche cache et d'une couche de sortie. La couche d'entre prend en entre les donnes de l'environnement (position et vitesse de la voiture) et les convertit en un vecteur d'entre. La couche cache est compose de plusieurs couches de neurones qui traitent les donnes et les convertissent en un vecteur de sortie. La couche de sortie prend en entre le vecteur de sortie de la couche cache et produit une action effectuer (acclrer, freiner ou ne rien faire).

import torch
import torch.nn as nn
import numpy as np

class DQN(nn.Module):
    def __init__(self, input_dim, output_dim):
        super(DQN, self).__init__()
        self.fc1 = nn.Linear(input_dim, 32)
        self.fc2 = nn.Linear(32, 32)
        self.fc3 = nn.Linear(32, output_dim)
        
    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = torch.relu(self.fc2(x))
        x = self.fc3(x)
        return x

La fonction de perte est une fonction qui mesure la diffrence entre les valeurs prdites et les valeurs relles. Dans le cas du DQN, la fonction de perte mesure la diffrence entre la valeur prdite par le rseau de neurones et la valeur relle obtenue par l'environnement en utilisant la formule suivante :

La fonction de perte est donne par : $$L(\theta) = \mathbb{E}_{s_t,a_t \sim \rho(.)}[(Q_{\theta}(s_t,a_t) - y_t)^2],$$ o $\theta$ est le vecteur de paramtres du rseau de neurones, $\rho$ est la distribution de probabilit de l'tat et de l'action, $Q_{\theta}$ est la fonction de valeur estime par le rseau de neurones, et $y_t$ est la cible de la fonction de valeur.

$$y_t = \mathbb{E}_{s_{t+1},a_{t+1} \sim \rho(.)}[r(s_t,a_t) + \gamma \max_{a}Q_{\theta}(s_{t+1},a_{t+1})].$$

import typing as t
import gym
import tqdm
import torch
import torch.nn as nn
import torch.optim as optim
import torch.nn.functional as F
from collections import deque
import random


def compute_loss(
    model: nn.Module,
    state: torch.Tensor, 
    next_state: torch.Tensor,
    reward: torch.Tensor,
    action: torch.Tensor,
    done: torch.Tensor,
    gamma: float = 0.99,
    batch_size: int = 32
):
    """
    Compute the DQN agent loss
    """
    q_values = model(state)
    q_next_state = model(next_state)
    q_values_target = reward + (1 - done) * gamma * q_next_state.max(dim=1)[0]
    loss = F.mse_loss(q_values[range(batch_size), action], q_values_target)
    return loss

# DQN Agent
class DQNAgent:
    def __init__(self, env, device):
        self.env = env
        self.device = device
        self.state_size = env.observation_space.shape[0]
        self.action_size = env.action_space.n
        self._memory: t.Deque[t.Tuple[np.ndarray, int, float, np.ndarray, bool]] = deque(maxlen=2000)
        self.batch_size = 32
        self.gamma = 0.95    # discount rate
        self.epsilon = 1.0  # exploration rate
        self.epsilon_min = 0.01
        self.epsilon_decay = 0.995
        self.learning_rate = 0.001
        self.model = self._build_model()

    def _build_model(self):
        # Neural Net for Deep-Q learning Model
        model = DQN(self.state_size, self.action_size).to(self.device)
        return model

    def remember(self, state: np.ndarray, action: int, reward: float, next_state: np.ndarray, done: bool):
        self._memory.append((state, action, reward, next_state, done))

    def act(self, state: np.ndarray) -> int:
        # Exploration : choisir une action alatoire avec une probabilit epsilon
        if np.random.rand() <= self.epsilon:
            return self.env.action_space.sample()
        else:
            q_values = self.model(torch.from_numpy(state).float().to(self.device))
            return torch.argmax(q_values).item()
        
    def replay(self):
        minibatch = random.sample(self._memory, self.batch_size)
        states = torch.from_numpy(np.vstack([x[0] for x in minibatch])).float().to(self.device)
        actions = torch.from_numpy(np.vstack([x[1] for x in minibatch])).long().to(self.device)
        rewards = torch.from_numpy(np.vstack([x[2] for x in minibatch])).float().to(self.device)
        next_states = torch.from_numpy(np.vstack([x[3] for x in minibatch])).float().to(self.device)
        dones = torch.from_numpy(np.vstack([x[4] for x in minibatch])).float().to(self.device)


        # Optimisation du rseau de neurones
        loss = compute_loss(self.model, states, next_states, rewards, actions, dones, self.gamma, self.batch_size)
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()

        # Rduction d'epsilon
        self.epsilon = max(self.epsilon_min, self.epsilon * self.epsilon_decay)


    def train(self, episodes: int):
        self.optimizer = optim.Adam(self.model.parameters(), lr=self.learning_rate)
        scores = []
        with tqdm.tqdm(range(episodes)) as t:
            for e in t:
                state, _ = self.env.reset()
                done = False
                score = 0
                while not done:
                    action = self.act(state)
                    next_state, reward, done, _, _ = self.env.step(action)
                    self.remember(state, action, reward, next_state, done)
                    state = next_state
                    score += reward
                    if len(self._memory) > self.batch_size:
                        self.replay()
                scores.append(score)
                t.set_postfix(reward=score)
        return scores

Double DQN¶

Le Double DQN est un algorithme d'apprentissage par renforcement qui combine le DQN avec une technique appele Prise de dcision double. Le Double DQN permet l'agent d'explorer plus intelligemment des environnements complexes et imprvisibles. L'algorithme est conu pour maximiser la valeur estime sur la base des informations disponibles.

L'algorithme commence par initialiser des paramtres pour le modle de rseau neuronal profond et deux cibles de rseaux distincts, X et Y. Les deux cibles sont des copies parallles, mais le modle X est mis jour plus frquemment que le modle Y. Quand un agent prend une action, il utilise le rseau X pour calculer la valeur estime et recueille le feedback aprs l'action. L'agent utilise ensuite le rseau Y pour dterminer quelle tait l'action optimale partir de cette valeur estime.

Cette procdure est appele prise de dcision double. Elle rduit l'instabilit et les biais de l'exploration lis la maximisation prmature.

from collections import deque, namedtuple
import torch.nn.functional as F


class DDQNAgent:
    def __init__(self, env, device):
        self.env = env
        self.device = device
        self.state_size = env.observation_space.shape[0]
        self.action_size = env.action_space.n
        self._memory: t.Deque[t.Tuple[np.ndarray, int, float, np.ndarray, bool]] = deque(maxlen=2000)
        self.batch_size = 32
        self.gamma = 0.95    # discount rate
        self.epsilon = 1.0  # exploration rate
        self.epsilon_min = 0.01
        self.epsilon_decay = 0.995
        self.learning_rate = 0.001
        self.model = self._build_model()
        self.target_model = self._build_model()
        
    def _build_model(self):
        # Neural Net for Deep-Q learning Model
        model = DQN(self.state_size, self.action_size).to(self.device)
        return model

    def remember(self, state: np.ndarray, action: int, reward: float, next_state: np.ndarray, done: bool):
        self._memory.append((state, action, reward, next_state, done))

    def act(self, state: np.ndarray) -> int:
        # Exploration : choisir une action alatoire avec une probabilit epsilon
        if np.random.rand() <= self.epsilon:
            return self.env.action_space.sample()
        else:
            q_values = self.model(torch.from_numpy(state).float().to(self.device))
            return torch.argmax(q_values).item()
                
    def replay(self):
        minibatch = random.sample(self._memory, self.batch_size)
        states = torch.from_numpy(np.vstack([x[0] for x in minibatch])).float().to(self.device)
        actions = torch.from_numpy(np.vstack([x[1] for x in minibatch])).long().to(self.device)
        rewards = torch.from_numpy(np.vstack([x[2] for x in minibatch])).float().to(self.device)
        next_states = torch.from_numpy(np.vstack([x[3] for x in minibatch])).float().to(self.device)
        dones = torch.from_numpy(np.vstack([x[4] for x in minibatch])).float().to(self.device)
        
        q_values = self.model(states)
        q_next_state = self.model(next_states)
        q_target_next_state = self.target_model(next_states)
        q_expected = q_values.gather(1, actions)

        # Double DQN
        q_argmax = q_next_state.argmax(dim=1)
        q_max = q_target_next_state.gather(1, q_argmax.unsqueeze(1)).squeeze(1)

        # Calculer la cible
        q_target = rewards + (self.gamma * q_max * (1 - dones))

        # Optimisation du rseau de neurones
        loss = F.mse_loss(q_expected, q_target)
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()

        # Rduction d'epsilon
        self.epsilon = max(self.epsilon_min, self.epsilon * self.epsilon_decay)

    def train(self, episodes):
        self.optimizer = optim.Adam(self.model.parameters(), lr=self.learning_rate)
        scores = []
        for e in tqdm.tqdm(range(episodes)):
            state, _ = self.env.reset()
            done = False
            score = 0
            while not done:
                action = self.act(state)
                next_state, reward, done, _, _ = self.env.step(action)
                self.remember(state, action, reward, next_state, done)
                state = next_state
                score += reward
                if len(self._memory) > self.batch_size:
                    self.replay()
            scores.append(score)
            if e % 10 == 0:
                self.target_model.load_state_dict(self.model.state_dict())
